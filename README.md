# Age

![age](age.webp)

A utility for people who want a quick reminder of birthdays, ages, and how long remains until the next celebration.

## Usage

```bash
age 2000-09-04     # quick age preview, does not save anything
age                # open help
age help           # open help

age list
age add Anton 1999-07-21
age add            # opens an interactive form
age update Anton   # updates name and/or birthday
age update         # choose a saved person interactively
age delete         # choose who to delete from a list
age delete Anton   # delete by name with confirmation
```
